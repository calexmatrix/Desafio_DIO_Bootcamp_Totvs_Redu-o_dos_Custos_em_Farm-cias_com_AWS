RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS
Data: 02 de abril de 2026

Empresa: Abstergo Industrias Farmacêuticas
Responsável: Carlos A.

1. Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industrias Farmacêuticas, realizado por Carlos. O objetivo do projeto foi elencar três serviços AWS com a finalidade de realizar a diminuição de custos imediatos, aumentar a escalabilidade e garantir a segurança dos dados farmacêuticos.

2. Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em três etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

Etapa 1:

Nome da ferramenta: Amazon EC2 (Elastic Compute Cloud)

Foco da ferramenta: Computação escalável sob demanda.

Descrição de caso de uso: Migração dos servidores locais de gestão de estoque e ERP para a nuvem. A utilização de Reserved Instances (Instâncias Reservadas) permitirá uma redução de custos de até 72% comparado ao modelo on-premise, eliminando gastos com manutenção de hardware físico e energia.

Etapa 2:

Nome da ferramenta: Amazon S3 (Simple Storage Service)

Foco da ferramenta: Armazenamento de objetos seguro e de baixo custo.

Descrição de caso de uso: Armazenamento de backups de receitas digitais, documentos regulatórios e histórico de vendas. Utilizando políticas de Lifecycle (S3 Intelligent-Tiering), os dados menos acessados são movidos automaticamente para camadas de arquivamento mais baratas, reduzindo drasticamente o custo de storage.

Etapa 3:

Nome da ferramenta: Amazon RDS (Relational Database Service)

Foco da ferramenta: Banco de dados relacionais gerenciados.

Descrição de caso de uso: Hospedagem do banco de dados de clientes e fórmulas farmacêuticas. O RDS automatiza tarefas como backup, correções de software e replicação. Isso reduz o custo operacional da equipe de TI e evita perdas financeiras decorrentes de possíveis quedas de sistema (downtime).

3. Conclusão
A implementação das ferramentas na empresa Abstergo Industrias Farmacêuticas tem como esperado a redução de custos operacionais, maior segurança no tratamento de dados sensíveis e alta disponibilidade dos sistemas críticos. Isso aumentará a eficiência e a produtividade da empresa, permitindo que o foco permaneça no core business farmacêutico. Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
